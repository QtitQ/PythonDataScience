## MongoDB 快速使用

    1. 将mongodb压缩包解压之后，将bin目录加到环境变量中，也就是path目录下
    2. 创建一个存放数据的地方，比如在moonodb目录下创建data目录
    3. 在命令行执行mongo --dbpath 刚才创建data目录的路径 //这样就启动了mongodb服务器端
    4. 重新打开一个命令窗口，mongo localhost:27017 就能访问mongodb服务器
